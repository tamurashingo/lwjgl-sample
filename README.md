# lwjgl-sample

## first sample

### build

```
mvn package
```

### run

```
java -jar target/lwjgl-sample-0.0.1-SNAPSHOT-jar-with-dependencies.jar
```


run on mac

```
java -XstartOnFirstThread -jar target/lwjgl-sample-0.0.1-SNAPSHOT-jar-with-dependencies.jar
```
